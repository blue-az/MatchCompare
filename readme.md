# Project Title

Import tennis sensor data from multiple sensors to compare reference match

## Description

Imports from ZeppU, Apple Watch, and Pop and compares a specific match. Output graph is ZIQ vs PIQ. ZIQ is based on PIQ.

## Getting Started

python3 main.py
